## English

HarmonClaw is an innovative AI hardware project that brings the power of OpenClaw AI assistant to OpenHarmony devices. Our goal is to create an accessible, open-source AI hardware solution for everyone.

### Features

- 🤖 **AI Assistant**: Full OpenClaw capabilities on OpenHarmony
- 🔧 **Open Source**: Fully open-source hardware and software
- 🌐 **Multi-Model Support**: Compatible with various AI models (Kimi, Wenxin, Tongyi)
- 📱 **Hardware Ready**: Designed for ARM-based OpenHarmony devices
- 🚀 **Easy Deployment**: One-click installation scripts
