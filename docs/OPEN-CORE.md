# HarmonClaw Open Core 策略

## 双仓库模型

- **主仓库 (BSL 1.1)**：完整功能，商业使用需授权
- **社区版 (Apache 2.0)**：Change Date 后自动切换，或提前授权的生产使用

## Change Date

4 年后（自首次公开发布起）自动转为 Apache 2.0。

## 社区版包含

- 核心运行时（Governor / Butler / Architect / Viking）
- 标准技能（doc_perceiver, web_search, tts, 代理技能）
- API 与 Web 控制台
- 配置、部署、CI 脚本

## 商业授权

生产环境商业使用在 Change Date 前需联系 启觉科技 获取授权。
